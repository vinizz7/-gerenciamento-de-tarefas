# Gerenciador de Tarefas em Python

Este projeto foi desenvolvido como parte dos meus estudos de Back-End com Python.  
Ele é um sistema simples de gerenciamento de tarefas via terminal, utilizando JSON para armazenar e carregar dados.

## Funcionalidades

- Adicionar novas tarefas  
- Listar tarefas cadastradas  
- Marcar como concluída (opcional)  
- Persistência de dados com arquivo .json  
- IDs gerados automaticamente  
- Sistema funciona mesmo após fechar o programa  

## Objetivo do Projeto

Praticar:
- Lógica de programação  
- Manipulação de arquivos  
- Estrutura de funções  
- Fluxos básicos de sistemas Back-End  
- Persistência usando JSON  

## Tecnologias Utilizadas

- Python 3  
- JSON
